# ACE-Step UI - 日本語エディション

ACE-Step UI の日本ユーザー向けにローカライズと機能強化を施した特別バージョンです。
本アプリケーションは、**100% ローカル環境・完全オフライン**で動作するように設計されています。

---

## 🇯🇵 日本語エディションの特長

### 1. 完全日本語対応
UIの全てのボタン、メニュー、メッセージを日本語化しました。サイドバーの地球儀アイコンからいつでも日本語と英語を切り替え可能です。

### 2. J-POP プリセットシステム
日本の音楽シーンに最適化された6つのスタイルプリセットを搭載しています。
*   **シティ・ポップ**: 80年代の都会的で洗練されたサウンド
*   **アニソン (OP風)**: 疾走感のある力強い主題歌スタイル
*   **ボカロ風**: 高速で緻密な電子ポップ
*   **切ないバラード**: ピアノとストリングスが美しい泣ける旋律
*   **J-ロック / アイドルポップ**: 王道の国内ジャンルを網羅

### 3. テーマおまかせ機能 (New!)
「テーマおまかせ」ボタンをクリックすると、日本の四季や風景（「桜の舞う季節」「真夏の夜のドライブ」など）をテーマにした曲名とスタイルをランダムに提案します。

### 4. ボーカル設定 (New!)
「男性」「女性」「ボカロ風」のボタンをクリックするだけで、最適なボーカルプロンプトを自動挿入します。

### 5. 完全オフライン動作
外部のCDNやスクリプト配信サービスを一切使用せず、全てのスタイルやプログラムをローカルに同梱しています。インターネット接続がない環境でも、プライバシーを保ちながら安心して使用できます。

---

## 🚀 起動方法

1.  **`launch_ja.bat`** (推奨) または **`日本版を起動.bat`** をダブルクリックします。
2.  初回起動時に、ACE-Step 1.5（エンジン側）が見つからない場合は、自動的に周辺フォルダをスキャンして接続します。
3.  APIサーバー、バックエンド、フロントエンドが順次立ち上がり、自動的にブラウザが開きます。

---

## 💾 ポータブル版のダウンロードと自動セットアップ

ポータブル版（`portable` フォルダ）を使用すると、さらに簡単にセットアップが可能です。

1.  `start.bat` を起動します。
2.  エンジンがない場合、`[1] 自動ダウンロード` を選択すると、約5GBの本体パッケージをその場でダウンロードします。
3.  ダウンロード完了後、案内通りに解凍・配置するだけで準備完了です。

## 💻 CPU環境（グラボなし）での動作

本バージョンは、**NVIDIA製グラフィックボードがないPC（内蔵グラフィックス等）でも快適に動作するように最適化されています。**

### 自動最適化機能
*   **自動CPUモード**: 起動時に `--device cpu` を自動適用するため、手動設定は不要です。
*   **爆速生成設定**: デフォルトの生成ステップ数を **4ステップ** に設定。CPUでも数分〜10分程度で1曲が完成します。
*   **パス自動解決**: ポータブル版エンジン（`ACE-Step-UI-JP-Portable`）を自動検出し、複雑なパス設定なしで動作します。
*   **タイムアウト延長**: CPUの低速な処理に対応し、最大2時間の生成を許容するように調整済みです。

### 推奨設定
*   **Turboモデル**: デフォルトで `1.5TS3 (Turbo)` が選択されています。そのまま使用することで最速の生成が可能です。
*   **ステップ数**: さらに高速化したい場合は、UIの「詳細設定」からステップ数をさらに下げることも可能です。

## 🛠 開発者向け情報

本プロジェクトは Vite + React + Tailwind CSS で構成されています。

```bash
# 依存関係のインストール
npm install
cd server && npm install

# 開発用起動
npm run dev
```

ビルド済みのファイルは `dist/` フォルダに生成され、オフラインでの配布が可能です。
