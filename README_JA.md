# ACE-Step UI - 日本語エディション

ACE-Step UI の日本ユーザー向けにローカライズと機能強化を施した特別バージョンです。
本アプリケーションは、**100% ローカル環境・完全オフライン**で動作するように設計されています。

---

## 🇯🇵 日本語エディションの特長

### 1. 完全日本語対応
UIの全てのボタン、メニュー、メッセージを日本語化しました。サイドバーの地球儀アイコンからいつでも日本語と英語を切り替え可能です。

### 2. J-POP プリセットシステム
日本の音楽シーンに最適化された6つのスタイルプリセットを搭載しています。
*   **シティ・ポップ**: 80年代の都会的で洗練されたサウンド
*   **アニソン (OP風)**: 疾走感のある力強い主題歌スタイル
*   **ボカロ風**: 高速で緻密な電子ポップ
*   **切ないバラード**: ピアノとストリングスが美しい泣ける旋律
*   **J-ロック / アイドルポップ**: 王道の国内ジャンルを網羅

### 3. テーマおまかせ機能 (New!)
「テーマおまかせ」ボタンをクリックすると、日本の四季や風景（「桜の舞う季節」「真夏の夜のドライブ」など）をテーマにした曲名とスタイルをランダムに提案します。

### 4. ボーカル設定 (New!)
「男性」「女性」「ボカロ風」のボタンをクリックするだけで、最適なボーカルプロンプトを自動挿入します。

### 5. 完全オフライン動作
外部のCDNやスクリプト配信サービスを一切使用せず、全てのスタイルやプログラムをローカルに同梱しています。インターネット接続がない環境でも、プライバシーを保ちながら安心して使用できます。

## 🚀 起動方法

1.  **`日本版を起動.bat`** をダブルクリックします。
2.  初回起動時に、ACE-Step 1.5（エンジン側）のフォルダパスを聞かれるので、フォルダをウィンドウにドラッグ＆ドロップして Enter を押してください。
3.  自動的にブラウザが立ち上がり、生成画面が表示されます。

---

## 💻 CPUのみでの動作について

GPU（NVIDIA製グラフィックボード）がない環境でも、CPUのみで動作させることが可能です。

### 設定方法
1.  **`日本版を起動.bat`** を右クリックして「編集」を選択します。
2.  以下の行を探します：
    `set API_COMMAND=python_embeded\python.exe -m acestep ...`
3.  末尾に ` --device cpu` を追加して保存します。
    例：`... --server-name 127.0.0.1 --device cpu`

### CPU利用時のヒント
*   **Turboモデル推奨**: 画面上のモデル選択から `1.5TS3 (Turbo)` を選ぶと、少ないステップ数で生成できるためCPU負荷を抑えられます。
*   **ステップ数の調整**: 「詳細設定 (Advanced)」から `Inference Steps` を **8〜10** 程度に下げると生成時間が短縮されます。

## 🛠 開発者向け情報

本プロジェクトは Vite + React + Tailwind CSS で構成されています。

```bash
# 依存関係のインストール
npm install
cd server && npm install

# 開発用起動
npm run dev
```

ビルド済みのファイルは `dist/` フォルダに生成され、オフラインでの配布が可能です。
